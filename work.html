Selected Projects

Lagrangian Dynamics of a Series Elastic Manipulator
This project applies Lagrangian dynamics and some of the structures in Geometric Mechanics to model and control the dynamics of a manipulator that has series elastic actuators. The purpose of this project was to explore the idea of traditional manipulators with tunable compliance, which could permit working in both industrial applications and alongside people.

Manipulator Path Planning Under Uncertainty As a class project with two other robotics students (and as a follow-on to the above work), I explored kinematic path planning for a manipulator with uncertainty in the actuators. This amounted to generating an uncertainty metric (the covariance of the actuators) in the joint space, which was then projected into the task space to act as a heuristic for path planning.

3D Rotational Coordinate Optimization
As part of an extended project, we're interested in producing dynamic legged locomotion (think running, jumping) on Agility Robotics' Cassie. My role in this project built on some previous work by my advisor, Prof. Ross L. Hatton, to optimize our choice of rotational coordinates for this robot.

Drawn Game Levels using Image Segmentation
This project grew out of a sentiment that traditional level design for videogames interrupted the creative process. In response, I wrote a tool that turns drawings into playable game levels. The application recognizes individual items in the drawing, and formats everything into an asset format recognizable by the Godot game engine.

Model-Based Control of McKibben Actuators
As part of a graduate course in Actuator Dynamics, I modeled the dynamic behavior of McKibben actuators, and investigated open- and closed-loop control approaches. These "air muscles" have some interesting robot applications, but effective control is still a challenge today.

Procedural Generation of Stories
While procedural generation receives a lot of attention in modern videogames, procedurally generated games often feel "empty." I wanted to address one potential reason, which is the lack of narrative. This project composes a set of related vignettes (story elements) and constructs a story graph to keep track of elements (characters, settings, items) in the story.

Wine Classification
As a project in one of my undergraduate courses, I designed a machine learning approach that classifies wine quality based on chemical properties. The dataset derived from prior work, but my method was able to provide comparable classification by reducing the resolution of measured quality. The full results are detailed in this (unpublished) paper.

Spectral Analysis
For my undergraduate capstone, I wrote another unpublished paper with a group of peers, exploring signal processing and audio compression. The project expanded my understanding of the Fourier transform, in its many forms.
